# LangChain

> **一句话总结**：LangChain是一个帮你把大语言模型和其他工具、数据、API"粘"在一起的开发框架，让你能像搭乐高积木一样组装自动运行的"思考链"。

---

## 🧩 核心定位

LangChain 的本质是一个**编排框架**，它提供了一套标准化的接口和组件，让开发者能够轻松地将 LLM 与各种外部资源连接起来。

## 🔗 主要功能

| 组件 | 功能 |
|------|------|
| **Chains** | 串联多个工具形成工作流 |
| **Agents** | 让模型自主决定调用哪个工具 |
| **Tools** | 封装外部API和服务 |
| **Retrievers** | 从数据库/文档中检索信息 |
| **Memory** | 管理对话历史和上下文 |

## 💡 典型应用场景

1. **智能客服**：结合知识库检索，提供精准回答
2. **数据分析**：连接数据库，自动生成SQL查询
3. **代码助手**：调用代码解释器，自动运行和调试代码
4. **工作流自动化**：串联多个工具完成复杂任务

---

**核心价值**：降低了构建复杂AI应用的门槛，让开发者专注于业务逻辑而非底层集成。
