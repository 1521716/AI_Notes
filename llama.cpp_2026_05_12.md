# llama.cpp

> 用C/C++实现的高效LLM推理框架，让大模型在消费级硬件上流畅运行。

---

## 核心定位

llama.cpp 是一个**推理框架**，专注于在本地高效运行大语言模型。它解决了"有了GGUF模型文件后，如何高效运行"的问题。

---

## 🔧 技术特点

| 特点 | 说明 |
|------|------|
| **纯C/C++实现** | 无Python依赖，跨平台支持 |
| **内存映射（mmap）** | 模型文件不必全部加载到内存，按需读取 |
| **CPU优先** | 专注CPU推理优化，也支持GPU加速（CUDA/Metal） |
| **GGUF原生支持** | 直接加载GGUF格式模型 |
| **批量推理优化** | 同时处理多个推理请求 |

---

## ⚙️ 工作原理

```
用户请求 → llama.cpp → 加载GGUF模型 → CPU/GPU推理 → 返回结果
```

1. **加载模型**：通过mmap将GGUF文件映射到虚拟内存
2. **分片加载**：大模型可分割成多个文件，按需加载
3. **推理计算**：利用CPU的AVX/AVX2/AVX-512指令集加速矩阵运算
4. **返回结果**：流式输出，边计算边返回

## ✅ 为什么重要

- **本地部署**：无需联网，保护隐私
- **低成本**：用普通电脑运行大模型
- **可定制**：开源可修改，可嵌入其他应用
- **高性能**：C语言级优化，推理速度快
