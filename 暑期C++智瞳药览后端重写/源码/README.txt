# 智瞳药览·明眼读药

> 基于多模态 AI 的老年人药品说明书智能解读与语音交互系统

## 项目结构

```
源码/
├── 前端/          # 微信小程序（用户端）
├── 服务器端/       # Flask 后端（部署于微信云托管）
└── README.md      # 本文件
```

## 技术架构

| 层 | 技术选型 | 说明 |
|----|---------|------|
| 前端 | 微信小程序 | 拍照上传、语音交互、结果展示 |
| 云函数 | 微信云开发 | baiduASR（语音识别）、callDeepSeek（AI 分析） |
| 后端 | Flask + 云托管 | OCR 识别（部署于微信云托管 Docker） |
| OCR | PaddleOCR | 药盒文字区域检测与识别 |
| AI | DeepSeek API | 药品说明解读、用药提醒生成 |
