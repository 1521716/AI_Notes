# 前端目录

微信小程序主目录，包含小程序源码、云函数、工具函数。

## 目录结构

```
前端/
├── app.js              # 小程序全局入口
├── app.json            # 全局配置（页面路由、窗口样式）
├── app.wxss            # 全局样式
├── cloudfunctions/     # 云函数（语音识别 + AI 调用）
├── pages/              # 小程序页面
├── utils/              # 工具函数
├── sitemap.json        # 微信 sitemap 配置
└── README.md           # 本文件
```
